# DataFusion
Ruby wrapper for [Apache Arrow DataFusion](https://arrow.apache.org/datafusion/).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'datafusion'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install datafusion
